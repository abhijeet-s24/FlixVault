# 🎬 FlixVault

## 📌 Project Overview
FlixVault is a dynamic web application that allows users to explore and discover movies in an interactive and user-friendly way. The application fetches real-time movie data from a public API and displays it dynamically on the webpage.

This project demonstrates the use of JavaScript, API integration, and modern UI design principles.

---

## 🎯 Features
- 🔍 Search movies by title
- 🎬 View movie details (poster, release year, ratings, etc.)
- 🎯 Filter movies by type or year *(planned)*
- 📊 Sort movies based on rating or release year *(planned)*
- 🌙 Dark/Light mode toggle *(planned)*
- ⏳ Loading indicator during API calls *(planned)*
